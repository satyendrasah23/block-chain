# Assignment 2: Smart Contract Deployment on Polygon Testnet (Amoy)

## Objective

The objective of this assignment is to deploy the SimpleStorage smart contract on the Polygon Amoy Testnet using Remix IDE and MetaMask wallet.


## Tools and Technologies Used

* Solidity
* Remix IDE
* MetaMask Wallet
* Polygon Amoy Testnet
* Polygonscan Explorer


## Network Details Used

Network Name: Polygon Amoy Testnet
RPC URL: https://rpc-amoy.polygon.technology
Chain ID: 80002
Currency Symbol: POL
Block Explorer: https://amoy.polygonscan.com


## Steps to Deploy Smart Contract on Polygon Amoy Testnet

Step 1:
Opened Remix IDE in browser.

Step 2:
Created SimpleStorage smart contract file (contract.sol).

Step 3:
Compiled the smart contract using Solidity compiler version 0.8.x.

Step 4:
Connected MetaMask wallet with Remix IDE using Injected Provider.

Step 5:
Selected Polygon Amoy Testnet network in MetaMask.

Step 6:
Requested free test POL tokens from Polygon faucet.

Step 7:
Deployed the smart contract using the Deploy button in Remix IDE.

Step 8:
Confirmed the deployment transaction from MetaMask.

Step 9:
Verified the transaction on Polygon Amoy Explorer (Polygonscan).
